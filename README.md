**Progress Tracker App**
 **Description:**
This is a progress tracking web application designed to help users manage tasks while visually monitoring their productivity. Unlike a basic to-do list, this app integrates performance tracking features such as streaks, completion statistics, and motivational messages to serve as encouragement.
**Features:**
- Add and manage daily tasks
- Mark tasks as completed
- Track completed vs pending tasks
- Streak counter to encourage consistency
- Visual progress tracking
- Motivational messages based on productivity
**Technologies Used:**
- HTML
- CSS
- JavaScript.
**Live Demo:**
[View Live Site](https://naomi12-chi.github.io/discipline-trackers/)
**How It Works:**
Users can create tasks and mark them as completed. The app automatically updates progress metrics, including total tasks created, completed tasks, and streak count, providing a visual representation of productivity.
